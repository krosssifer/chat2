Клонируйте репозиторий: git clone https://github.com/krosssifer/angular-chat.git](https://github.com/krosssifer/chat2)
Перейдите в директорию проекта: cd (path)./cha2-main
Установите зависимости: npm install
Запустите приложение: npm run start
Откройте браузер и перейдите по адресу http://localhost:4200 Для тестирования второго пользователя смените имя пользователя во второй вкладке
